# House Of Brands Showcase

Minimal static showcase website for House Of Brands.

## Run locally

```powershell
node server.mjs
```

Then open `http://localhost:4173`.

## Structure

- `index.html` - page content and sections
- `styles.css` - visual system, responsive layout and animations
- `script.js` - hero rotation, scroll reveal and progress behavior
- `assets/` - copied HOB, partner, category and campaign imagery
